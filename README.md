# 🌐 Responsive Web Design & Development Course Website

A responsive website project created by **IRSHA** to showcase a *Web Design & Development Course*.  
The design features a modern UI with a serene background, intuitive navigation, and interactive login functionality.  
This project demonstrates responsive web design principles, ensuring smooth performance across devices.

---

## ✨ Features
- **Navigation Bar**: Links to Home, About, Service, Design, and Contact  
- **Search Bar**: Styled input field with search button  
- **Course Promotion Section**: Highlighting the course with a clear call-to-action (*Join Us*)  
- **Login Form**: Email & password fields, login button, and sign-up option for new users  
- **Social Media Integration**: Facebook, Instagram, Twitter, Google, Skype icons for quick access  
- **Responsive Layout**: Works seamlessly on desktop, tablet, and mobile  

---

## 🛠️ Tech Stack
- **HTML5** → Page structure  
- **CSS3** → Styling and responsiveness  
- **JavaScript** → Optional interactivity

- Smooth scaling across different screen sizes
